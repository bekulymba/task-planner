# Task Planner

## О проекте
**Task Planner** -  это веб-приложение для управления задачами, реализованное на чистом JavaScript без использования фреймворков и библиотек.  
Проект демонстрирует работу с DOM, обработку ошибок и реализацию пользовательских функций интерфейса.  
Пользователь может создавать задачи, редактировать их, изменять статус и управлять списком задач через удобный интерфейс.  
Каждая задача содержит:  
- уникальный ID
- время создания
- название
- описание
- статус задачи

Статусы задач:  
- Active - активная задача
- Completed - выполненная задача
- Failed - невыполненная задача

Все задачи сохраняются локально в браузере с использованием LocalStorage в формате JSON.

## Основные возможности
- создание задач
- редактирование названия и описания
- удаление задач
- изменение статуса задачи
- Drag & Drop задач
- Multi-selection задач

фильтрация и сортировка
## Стандарты качества
Проект соответствует следующим требованиям: 
1. HTML проходит валидацию W3C  
2. CSS проходит валидацию W3C  
3. JavaScript соответствует правилам ESLint  
4. Приложение работает во всех современных браузерах  
5. В консоли отсутствуют ошибки    

## Проверка
**Проверка Eslint**
Запустите команду в корневой папке проекта:
``` bash
npx eslint .
```
Если ошибок нет - команда не выведет сообщений.  

**Проверка HTML**
Используйте валидатор:
``` bash
https://validator.w3.org/
```
**Проверка CSS**
Используйте валидатор:
``` bash
https://jigsaw.w3.org/css-validator/
```
## Установка и запуск

Клонировать репозиторий
```bash
git clone https://github.com/bekulymba/task-planner
```
Открыть файл ```index.html``` в браузере.

Проект не требует установки зависимостей.

## Как пользоваться сайтом
1. **Чекбокс** - Выделение задачи.  
2. **Троеточие** - Редактирование задачи.  
3. **Крестик** - Удаление задачи.  
4. **Complete** - Отметить задачу как выполненную.  
5. **Fail** - Отметить задачу как невыполненную.
